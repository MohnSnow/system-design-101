# 🌐 必知的8种DNS记录类型！网络基础必修课

> 域名解析不只是A记录，还有这些你该知道的

![DNS记录类型](dns-record-types-you-should-know.png)

8种最常用的DNS记录类型 👇

📌 **A记录** — 域名→IPv4地址，最基础的记录

📌 **AAAA记录** — 域名→IPv6地址

📌 **CNAME记录** — 域名别名，常用于子域名指向主域名

📌 **PTR记录** — 反向DNS查找，IP→域名，用于验证服务器真实性

📌 **MX记录** — 邮件交换记录，指定邮件服务器

📌 **NS记录** — 指定域名的权威DNS服务器

📌 **SRV记录** — 指定特定服务的主机和端口（如VoIP）

📌 **TXT记录** — 文本记录，常用于SPF邮件安全验证

💡 日常开发中最常用的是A、CNAME和MX记录。配置域名时搞清楚这几个就够了。
