# ⏰ 人生苦短，快用这些开发神器！8大类工具推荐

> 选对工具，省下的时间够你过个好周末

![开发工具推荐](./life-is-short-use-dev-tools.png)

好的工具能让效率翻倍，这里按类别整理了我们最爱的开发工具 👇

📌 **开发环境**
VSCode、IntelliJ IDEA、Vim、PyCharm、Jupyter Notebook

📌 **画图工具**
DrawIO、Excalidraw、Mermaid、PlantUML、Miro

📌 **AI工具**
ChatGPT、GitHub Copilot、Claude、Ollama、Midjourney

📌 **部署托管**
AWS、Cloudflare、GitHub Pages、Fly、Heroku

📌 **代码质量**
Jest、ESLint、Selenium、SonarQube、Checkstyle

📌 **安全工具**
1Password、OWASP、Snyk、Nmap

📌 **笔记工具**
Notion、Obsidian、Markdown、Logseq

📌 **设计工具**
Figma、Sketch、Canva、Adobe Illustrator

💡 不用全部都用，每个类别选1-2个趁手的就够了。工具是为你服务的，别被工具绑架。

你最离不开的开发工具是哪个？我先说：VSCode + Copilot 👇
